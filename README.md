# Desafio DIO - Configuração de Banco de Dados no Microsoft Azure

## Objetivo

Este repositório foi criado para documentar os conhecimentos adquiridos durante o laboratório de configuração de Banco de Dados na plataforma Microsoft Azure.

## O que é o Microsoft Azure?

O Microsoft Azure é uma plataforma de computação em nuvem que oferece diversos serviços para criação, hospedagem e gerenciamento de aplicações e bancos de dados.

## Azure SQL Database

O Azure SQL Database é um serviço de banco de dados relacional baseado no SQL Server, disponibilizado na nuvem pela Microsoft.

### Principais vantagens

- Alta disponibilidade
- Escalabilidade
- Backup automático
- Segurança integrada
- Gerenciamento simplificado

## Conceitos Aprendidos

### Resource Group (Grupo de Recursos)

É um contêiner utilizado para organizar e gerenciar recursos relacionados dentro do Azure.

### SQL Server

Responsável por hospedar e gerenciar os bancos de dados criados na plataforma Azure.

### Banco de Dados SQL

Permite armazenar e gerenciar informações utilizando a tecnologia SQL Server sem a necessidade de administrar infraestrutura física.

### Segurança

Algumas configurações importantes:

- Usuário administrador
- Senhas fortes
- Regras de Firewall
- Controle de acesso

## Aprendizados

Durante o laboratório foi possível compreender:

- Conceitos de computação em nuvem;
- Estrutura básica dos serviços do Azure;
- Funcionamento do Azure SQL Database;
- Importância da segurança em ambientes de nuvem;
- Organização de recursos utilizando Resource Groups.

## Conclusão

O laboratório apresentou os conceitos fundamentais sobre bancos de dados em nuvem utilizando o Microsoft Azure, servindo como base para estudos futuros e implementações práticas.
